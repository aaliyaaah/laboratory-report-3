## Introduction

These experiments explore different digital modulation techniques used in modern communication systems. They demonstrate how digital data can be transmitted by modifying characteristics of a carrier signal such as amplitude, frequency, and phase, and how the original information can be recovered using different demodulation and signal restoration methods.

---

### Experiment 15: Amplitude Shift Keying
This experiment demonstrates how digital information can be transmitted by varying the amplitude of a carrier signal. It also shows how the modulated signal can be detected and restored to its original digital form.

Part A - Generating an ASK signal  
Part B - Demodulating an ASK signal using an envelope detector  
Part C - Restoring the recovered digital signal using a comparator  

---

### Experiment 16: Frequency Shift Keying
This experiment shows how digital data can be represented using different carrier frequencies and how the signal can be recovered using filtering and detection techniques.

Part A - Generating an FSK signal  
Part B - Demodulating an FSK signal using filtering and an envelope detector  
Part C - Restoring the recovered data using a comparator  

---

### Experiment 17: Binary Phase Shift Keying
This experiment demonstrates how digital information can be transmitted by changing the phase of the carrier signal and recovered using phase detection methods.

Part A - Generating a BPSK signal  
Part B - Demodulating a BPSK signal using a product detector  
Part C - Restoring the recovered data using a comparator  

---

### Experiment 18: Quadrature Phase Shift Keying
This experiment introduces QPSK modulation, where two bits of information are transmitted per symbol using different carrier phase states.

Part A - Generating a QPSK signal  
Part B - Using phase discrimination to pick-out one of the QPSK signal’s BPSK signals  

---

### Experiment 19: DSSS Modulation and Demodulation
This experiment explores direct sequence spread spectrum techniques and demonstrates how signals can be spread over a wide bandwidth and later recovered.

Part A - Generating a DSSS signal using a simple message  
Part B - Generating a DSSS signal using speech  
Part C - Using the product detector to recover the message  
Part D - DSSS and deliberate interference (jamming)  

---

### Experiment 20: Undersampling in Software Defined Radio
This experiment demonstrates how undersampling techniques can be used in software defined radio systems for signal conversion and recovery.

Part A - Setting up a bandwidth limited signal  
Part B - Direct down-conversion using undersampling  
Part C - Synchronisation
